private access without a vpn

github repo run with npx

derphole

move a file

derphole is the one-session path for files, streams, and localhost shares.

npx -y derphole@latest receive <code>

derptun

open a tcp tunnel

Put any TCP service behind a durable tunnel, then share access with client tokens or embed it in an app. 

npx -y derptun@latest token server > server.dts
npx -y derptun@latest serve --token-file server.dts --tcp 127.0.0.1:3000

derpssh

share your terminal

derpssh is terminal sharing over the same no-open-port transport.

npx -y derpssh@latest connect <invite>

not a vpn

scoped access, not a network

Scoped tokens authorize one transfer, one tunnel, or one terminal session. No daemon, overlay interface, tailnet, subnet route, or account-backed control plane.

derp explained

direct paths when possible

DERP is Tailscale's Designated Encrypted Relay for Packets. Tailscale operates a public DERP network for rendezvous and fallback; derphole uses it so peers can find each other when both sides can only dial out. Sessions prefer direct paths when available. Payloads stay end-to-end encrypted, and this project is not affiliated with Tailscale.

one transport family

small tools, shared path

derphole handles one-shot transfers and temporary service shares. derptun keeps TCP services reachable with persistent tunnels. derpssh shares a shell over the same no-open-port transport.
